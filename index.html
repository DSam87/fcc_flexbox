FlexBox Overview!

Going Over Flexbox Containers And Flexbox Items

Flex Container & It's Properties

display: flex

This element is placed on a container allowing us to use other flex properties and creates flex items.

flex-direction

flex-direction property on the flex container to choose the main axis direction

flex-direction Values

  • column
  • row
  • column-reverse
  • row-reverse

justify-content

The Items within a flex container can be told how to align & space out items in a certain way.

This does thin along the main axis.

justify-content Values

  • column
  • row
  • column-reverse
  • row-reverse

align-items

For the cross axis (which is the opposite of the main axis).

To align flex items along the cross axis.

align-elements Values

  • flex-start
  • flex-end
  • center
  • stretch
  • baseline

flex-wrap

Property tells CSS to wrap items.

This does thin along the main axis.

align-elements Values

  • nowrap
  • wrap
  • wrap-reverse

Flex Items & It's Properties

flex-shrink


It controls the size of the items specificly when the container shrinks.


flex-shrink property takes a number as values, the height the number the more it shrinks compared to the other items in the container.

flex-grow

flex-grow property controls the size of items when the parent container expands.

flex-basis


flex-basis property specifies the intial size of the item before CSS makes adjustments with flex-shirnk or flex-grow.


flex-basis can use many units the same as other size proeprties (px,em,%)

flex Shorthand

flex-grow, flex-shrink, and flex-basis properties can all be set together by using the flex property.

order

The order property is used to tell CS the order of how flex items apear in the flex container.

align-self

Allows us to adjust each item's alignment individually,instead of setting them all at onece. (with align-items)

align-self Values

  • flex-start
  • flex-end
  • center
  • stretch
  • baseline
